Instalación de Ldap cliente

  1. Instalamos los paquetes libnss-ldap

  2. Instalamos la libreria libpam-ldap

  3. Instalamos el paquee nscd

  4. Modificamos el archivo /etc/ldap.conf

  5. Iniciamos el asistente de instalación de nuevo con el comando: dpkg-reconfigure ldap-auth-config

  6. Indicamos la URIo dirreción para acceder a ldap

  7. Seleccionaos la versión de Ldap que estamos utilizando

  8. Seleccionamos que si, para que se pueda cambiar las contraseñas del usuario del directorio más adelante

  9. Damos que no es necesario autentificarse para consultar el directorio LDAP

  10. Indicamos a cuenta de administrador del servidor LDAP

  11. Indicamos la contraseña de admin, que en nuestro caso es root

  12. Configuramos el archivo /etc/nsswitch.conf

  13. Configuramos los archicos common-auth

  14. Configuramos los archivos common-account

  15. Configuramos el archivo common-session

  16. Configuramos el archivo common-password

  17. Añadimos al final del fichero common-session las siguientes lineas
