System V

Script Demonio

  1. Creamos el Script y lo mentemos en /usr/local/bin/kupd

  2. Damos permisos de ejecución con el comando chmod 777

  3. Instalamos Ruby

  4. Creamos el directorio /etc/kup con el comando mkdir

  5. Ejecutamos el Script y comprobamos que funciona haciendo cat en el archivo cat /etc/kup/kup.log

  6. Lo paramos ejecutando el comando rm /etc/kup/running

Script de control

  1. Creamos el archivo en /etc/init.d/kup

  2. Damos permisos de ejecución con el comando chmod 777

  3. Probamos que funciona mediante el uso de Start|Stop|Status

  4. Miramos nuestro runlvel, que suele ser el Nº2

  5. Creamos un enlace simbolico en /etc/rc2.d con el nombre S99kup

  6. damos permisos de ejecución en /etc/rc2.d

  7. Modificamos el Script para que se pueda arrancar sin problemas

  8. Vemos que se nos ejecuta sin problemas al iniciar el sistema
