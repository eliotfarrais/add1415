Instalación y configuración

  1. Instalamos y buscamos los siguientes paquetes desde Synaptic:
    apt-get install ltsp-server-standalone

    Nota: Openssh viene ya instalado

  2. Configuramos las tarjetas de red, una estatica para la red internet(eth1) y otra para la red interna donde va nuestro cliente(eth0)

  3. Configuramos las tarjetas de red interna en virtual box con el adaptador de red PCnet-FAST III para servidor y cliente

  4. Como orden de arranque, ponemos red como la primera en el cliente

Funcionamiento

Como vemos se nos ejecuta el cliente ligero en el cliente, sin problemas
