Instalación y configuración de VNC

Instalación del Servidor Windows Server

  1. Nos descargamos RealVNC de: https://www.realvnc.com/download/get/1590/

  2. Instalamos el servidor de VNC. En la instalación elejimos todos los componentes

  3. Obtenemos la licencia Free desde la página del servidor

  4. Aceptamos la conexiones no cifradas en la instalación

  5. Ponemos la contraseña predeterminada, en nuestro caso será "root"

  6. Una vez realizada la instalación, se nos iniciará el servidor VNC. Se debe de inciar correctamente con todos lo Servicios en Activo

Instalación del Servidor Windows Cliente

  1. Nos vamos al cliente, e instalamos VNC viewer, el cliente del VNC para ver el escritorio remoto

  2. Una vez instalado, pondremos la ip o el nombre dns de nuestro servidor VNC.

  3. Aceptaremos las conexiones no cifradas entre cliente y servidor

  4. Pondremos nuesra contraseña predeterminada

  5. Nos hemos conectado correctamente a nuestro servidor VNC en el servidor

Instalación del Lubuntu Servidor

  1. Nos descargamos de la página oficial el Servidor para Linux de debian y lo instalamos

  2. Nos registramos y ponemos la licencia. Con lo que así registramos el programa

  3. Elejimos la conexiones no cifradas

  4. Como contraseña ponemos root

  5. Nos conectamos al Servidor Lubuntu

Instalación del Lubuntu cliente

  1. Nos descargamos el cliente Remmina para Lubuntu en Synaptics

  2. Lo abrimos y realizamos una conexión VNC, eligiendola en la parte protocolos.

  3. Se nos realiza la Conexión

Instalación y configuración de RDP

Instalación en Windows 2008 Server RDP

En el caso de Windows no es necesario instalar escritorio RDP, dado qe viene integrado con Windows

  1. Para configurarlo, en el servidor, Nos vamos a Menu Inicio -> Equipo -> Propiedades -> Configuración de acceso remoto -> Ejecutar cualquier versión

  2. Iniciamos escritorio remoto en el Cliente, y ponemos la dirreción Ip del Servidor

  3. Seleccionamos el nombre de usuario y la contraseña

  4. Se nos inicia sesión sin problemas en el Cliente Windows

  5. Se nos inicia sesión sin problemas en el Cliente Lubuntu con Remmina

Instalación en Lubuntu de Servidor rdp

  1. Instalamos el servidor xrdp desde Synaptics.

  2. Configuramos el archivo: sudo nano /etc/xrdp/startwm.sh y editamos

    #. /etc/X11/Xsession
    . /usr/bin/startlxde
    Quedando de la siguiente manera

  3. En el cliente, conectamos con la IP en el Servidor

    4. Cuando nos salga la siguiente ventana, elejiremos en modules vnc-any y pondremos la contraseña de superusuario

  4. Se nos conecta sin problemas al escritorio de Lubuntu con protocolo rdp

Instalación y configuración de Servidor de Terminales

  1. Abrimos el Administrador de servidores y vamos a Agregar Roles. DAmos clic en Agregar Servicios de escritorio remoto y damos clic en siguiente

  2. Dentro de los servicios de rol, activamos la casilla Host de sesión de Escritorio remoto y pulsamos siguiente

  3. Al elegir el Método de autentificación, elejimos "No requerir autenticficación de red"

  4. En el modo de licencia, elejiremos por dispositivo

  5. En el Grupo de usuarios, agregamos a Administrador para poder iniciar sesión con el

  6. Activaremos las Experiencias del cliente, activando todas sus opciones. Una vez terminado esto, le damos a instalar y reiniciamos el sistema.

  7. Pondremos los usuario dentro del grupo de escritorio remoto

  8. Iniciaremos sesión con el usuario que hemos añadido al grupo

Instalación y configuración de RemoteApp

  1. Añadimos las herramientas del host de escritorio remoto.

  2. Abrimos remoteapp

  3. Nos dirijimos a Configuración de Puerta de enlace de Escritorio remoto, y configuramos las conexiones de acceso remoto.
    una vez finalizado cerramos

  4. Añadimos una nueva alpicacion a remoteapp. Para ello nos dirijimos a Agregar Programas a remoteapp

  5. Agregamos en Wordpad el nuevo programa

  6. Terminamos agregando y dando a aceptar

  7. Iniciamos el programa desde el escritorio remoto, sin problemas
