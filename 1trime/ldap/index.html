instalación de Ldap

Instalación del Servidor Windows Server

  1. Instalamos los paquetes slapd db5.3-util ldap-utils

  2. Lanzamos el asistente de instalación de ldap con dpkg-reconfigure sldap

  3. Lo primero que nos pregunta es que si vamos a cerrar el asistente, le decimos que no

  4. Aquí ponemos el nombre de nuestro dominio

  5. Luego aquí pondremos el nombre de nuestra organización

  6. Pondremos luego aquí la contraseña del administrador

  7. Verificamos la contraseña

  8. Seleccionaremos el sistema BDB

  9. Le diremos que no a al ahora de purgar la base de datos

  10. Si nos pide mover los datos a una nueva base de datos, le diremos que si

  11. Nos preguntará para usar el protocolo LDAPv2. al que le diremos que no

    Al finalizar este paso hemos terminado todo el proceso de configuración de LDAP

  12. Ahora nos toca iniciar ldap, para ello pondremos el comando /etc/init.d/slapd stop | start

    13. instalación de phpLDAPadmin

    1. Descargaremos los paquetes "apache2", "libapache2-mod-php5", "php5-ldap" por Synaptic

    2. Instalaremos ahora phpLDAPadmin a través de Synaptic

    3. Si todo ha salido correctamente se nos habrá iniciado todo correctamente
