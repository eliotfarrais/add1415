Instalación NFS en Windows

  1. Instalamos con el comando apt-get los siguientes archivos:
    apt-get install -y samba samba-common smbclient samba-doc cifs-utils

  2. Guardamos el viejo fichero y creamos un nuevo de la configuracion de Samba

  3. Modificamos el fichero smb.conf

  4. Añadimos usuarios y grupos para samba

  5. Creamos directorios y cambiamos el usuario propietario para que quede como la imagen, con el comando chown

  6. Modificamos tambiñen los grupos con chgrp. Quedando algo así

  7. Añadimos los usuarios a samba

  8. Nos quedaría algo así

  9. Miramos el estado de samba, y sinó está iniciado el servicio, lo iniciamos
