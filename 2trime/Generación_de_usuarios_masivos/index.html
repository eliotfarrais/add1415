Servidores de impresión

Instalación Gwak para Windows

  1. Nos descargamos de la página oficial gwak y lo instalamos

  2. En la instalación elegimos Full installation

  3. Luego Seleccionamos Download Sources

  4. Metemos dentro de la variable PATH del sistema el directorio de instalación de Awk

Instalación de Script XCALCS.vbs

Es un script que nos permite, ver, modificar y eliminar permisos NTFS en Windows

  1. Descargamos el Script desde: http://download.microsoft.com/download/f/7/8/f786aaf3­a37b­45ab­b0a2­8c8c18bbf483/XCacls_Installer.exe y lo abrimos

  2. Buscamos la función IsOsSupported y la modificamos

  3. Cambiaremos el motor de secuencias de comandos predeterminado de WScript a CScript

Ficheros CSV y AWK

  1. Creamos el fichero CSV, creandolo a mano. Este mismo contiene cuatro campos: nombre;apellido1;apellido2;grupo

  2. Creamos el Script alta.awk

  3. Se nos generael sigiente archivo .bat

  4. Comprobamos el resultado, mirando dentro de la unidad organizativa FP., lo que se ha creado

  5. Comprobamos que se han creado los usuarios dentro de 1CFGM

  6. Comprobamos que se han creado los usuarios dentro de 2CFGM

  7. Miramos las carpetas creadas por el sistema
