Tareas programadas en Linux

Tareas Programadas diferidas (at)

  1. Ejecutar el apagado de la máquina con el comando.
    Para ello escribimos "at (hora de ejecución) y luego escribimos el comando "shutdown -h 1"(Apagado en un minuto).
    Por ultimos pulsamos CTRL + D en el teclado para guardar los cambios de at en el equipo

  2. Comprobamos las tareas pendientes con el comando atq

Tareas programadas periódicas (crontab)

  1. Mostrar un mensaje a pantalla
    Para ellos podemos usar el siguiente script:
    -----------------------------------------------------------------------
    #!/bin/bash
    # Mostrar mensaje en pantalla
    DISPLAY=:0
    export DISPLAY
    zenity --info --text="Tarea ejecutada"
    -----------------------------------------------------------------------


    Y lo creamos dentro una carpeta del path, en nuestro caso será "/usr/local/bin" con permisos 744

  2. Ejecutamos cron con el comando crontab -e (desde profesor) y editamos el archivo.
    Ponemos la ubicación, y que se ejecute a la hora deseada de todos los días

  3. Instalamos el paquete zenity para que pueda ser utilizado por el script

  4. la tarea se nos abrá ejecutado automáticamente

Tareas programadas asíncronas (anacron)

  1. Para el uso de tareas programadas asíncronas (anacron),
    instalaremos anacron primeramente

  2. Editamos el archivo /etc/anacrontab y agregramos la entrada del lanzador.
    Por ultimo obligamos la ejecución de anacron con anacron -n

Tareas programadas en Windows

  1. Abrimos el programador de tareas en
    Panel de control -> Sistemas y seguridad -> herramientas administrativas -> Programador de tarea.

  2. Creamos una tarea básica, donde pondremos el apagado del sistema

  3. Escribimos el nombre de apagado y la descripción de la tarea.

  4. Creamos una tarea Diariamente

  5. Elegimos los días, la hora y las veces que se va a repetir en los días de la semana

  6. Hacemos clic en Inicar un programa y elegimos el script de apagado que hemos realizado para apagar el sistema

  7. Por último damos finalizar y se nos habrá creado la tarea programada
    Nos saldrá la tarea programada en el programador de tareas.
